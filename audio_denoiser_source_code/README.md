# AI降噪演示 - ZegoAIDenoise算法实现

基于ZEGO即构科技博客文章《消灭非稳态噪音的利器 - AI 降噪》的算法原理实现。

## 项目概述

本项目演示了ZegoAIDenoise算法的核心原理，这是一个轻量级神经网络降噪方法，结合传统信号处理和深度学习技术，专门针对非稳态噪音进行降噪处理。

## 算法特点

- **Hybrid方法**：传统算法与深度学习相结合
- **频域子带处理**：使用22个巴克频带尺度
- **CRNN网络**：卷积层 + GRU层的混合架构
- **轻量级设计**：优化的网络结构，低性能开销

## 核心功能

1. **音频信号处理**：STFT变换、子带分解
2. **神经网络降噪**：基于CRNN的增益估计
3. **实时降噪演示**：交互式音频处理界面
4. **效果对比**：原始音频与降噪后音频对比

## 算法原理

基于博客中描述的ZegoAIDenoise算法：
1. 对输入音频进行STFT变换得到频域信号
2. 使用巴克频带尺度将频域分成22个子带
3. 使用CRNN网络估计每个子带的增益
4. 应用增益进行谱减法降噪
5. 通过逆STFT重构时域信号
